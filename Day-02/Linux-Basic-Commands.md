# Linux Basic Commands

## pwd

### Description:

The `pwd` command displays the current working directory.

### Example:

```bash
pwd
```

Output:

```text
/home/aynaz
```

---

## ls

### Description:

The `ls` command lists files and directories in the current directory.

### Example:

```bash
ls
```

Output:

```text
Documents Downloads notes.txt
```

---

## mkdir

### Description:

The `mkdir` command creates a new directory.

### Example:

```bash
mkdir Day2
```

Output:

```text
Day2
```

---

## cd

### Description:

Changes the current directory.

### Example:

```bash
cd Documents
```

---

## cp

### Description:

Copies files or directories.

### Example:

```bash
cp file.txt backup.txt
```

---

## mv

## Description:

Movess or rename files.

### Example:

```bash
mv file.txt Documents/
```

---

# rm

### Description:

Deletes files or directories.

### Example:

```bash
rm file.txt
```

---

# cat

### Description:

Displays the content of a file.

### Example:

```bash
cat notes.txt
```

---

## clear 

### Descriptin:
Clears thr terminal screen.

### Example:

```bash
clear
```
